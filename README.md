Healthcare Analytics Dashboard

Overview

This project is an interactive Power BI dashboard designed to analyze hospital operations, patient demographics, clinical trends, and financial performance. 
The dashboard transforms raw healthcare data into actionable insights that support data-driven decision-making.

Objectives
- Analyze patient demographics across different age groups, genders, and blood groups.
- Evaluate hospital financial performance using billing metrics.
- Identify the most common medical conditions and treatment patterns.
- Monitor admission trends and operational efficiency over time.
- Build an executive-style dashboard using Power BI and DAX.

Tools Used
- Power BI
- DAX
- Power Query
- Git & GitHub

Dashboard Features
Executive KPIs

Total Patients
- Total Billing Amount
- Average Billing Amount
- Average Length of Stay

Patient Analysis
- Age Group Distribution
- Gender Distribution
- Blood Group Distribution
- Insurance Provider Distribution

Clinical Analysis

- Medical Condition Distribution
- Average Billing by Medical Condition
- Average Length of Stay by Medical Condition
- Most Common Medical Conditions

Financial Analysis

- Total Billing by Hospital
- Total Billing by Insurance Provider
- Total Billing by Admission Type

Time-Series Analysis

- Monthly Patient Admissions
- Monthly Billing Trends
- Monthly Average Length of Stay

Key Insights

- Compare hospital revenue across facilities.
- Identify the most prevalent medical conditions.
- Understand patient demographics and insurance coverage.
- Track financial and operational trends over time.
- Evaluate hospital performance using interactive filters and visualizations.
